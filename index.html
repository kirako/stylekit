Full Action Buttons

  1. Colgroup: Add the "items-#" class to a column where # is the max number of items.
  2. Cells: Add the "full" class to th/td's with "actions" to make the buttons expand to all edges.
  3. Buttons: Add the "full" class to buttons to make the expand the full width of the cell.
